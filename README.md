Sau đây là một số câu lệnh thường dùng với Vim.

1. Lưu file - ":w"

2. Thoát khỏi file mà không lưu - ":q"

3. Lưu và thoát khỏi file - ":wq"

4. Chuyển sang chế độ insert - "i"

5. Chuyển sang chế độ command - ESC

6. Xóa một từ - "dw"

7. Xóa một dòng - "dd"

8. Quay trở lại các bước trước - "u".
